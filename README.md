Simple Chat Server

一个基于 Java 构建的简易分布式即时通讯系统（学习项目）。

通过从零开始构建一个聊天系统，巩固 Java 面向对象编程思想，学习了解：
- Java 面向对象建模
- Maven 项目管理和依赖管理
- SLF4J & Logback 日志框架
- Socket 网络编程
- 多线程并发处理
- 序列化与通信协议设计
- ...（后续持续更新）

- **语言**: Java 23
- **构建工具**: Apache Maven
- **日志框架**: SLF4J + Logback
